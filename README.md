# Webdrobe

Proje Amacı ve Kapsamı Günümüzde bireylerin sahip oldukları kıyafet ve eşyaların sayısı
arttıkça, bu eşyaların nerede bulunduğunu, hangi kategoride kaç tane olduklarını hatırlamak
giderek zorlaşmaktadır. Bu durum, özellikle fazla eşyaya sahip kullanıcılar için zaman
kaybına ve düzensizliğe yol açmaktadır.
Bu proje, kullanıcıların dolaplarındaki kıyafetleri dijital ortamda organize edebilmelerini
sağlayan bir “Wardrobe Management System” (Dolap Yönetim Sistemi) geliştirmeyi
amaçlamaktadır.
Kullanıcılar, sistem üzerinden kıyafetlerinin konumlarını manuel olarak belirleyebilecek,
kombinler oluşturabilecek ve arama/filtreleme özellikleri sayesinde istedikleri eşyalara
kolayca ulaşabilecektir.


#Bağımlılıkları yükleyin:

```
dart pub get
```

#Çalıştırın:

```
dart run
```
